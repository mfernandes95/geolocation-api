# CityClient-api

- NodeJs API using Express
- Database using MongoDB
- TDD using Jest
- Containerization using Docker
- Documentation using Swagger
- CI using GitHub Actions

## Run

```bash
sudo docker-compose up --build
```

## Get Documentation

http://localhost:3333/api-docs/
